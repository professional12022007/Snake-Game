# 🐍 Snake Game

A classic Snake game built with pure HTML, CSS, and JavaScript — no libraries, no dependencies. Just open the file in a browser and play.

---

## 🎮 Demo

Open `snake_game.html` in any modern browser to play instantly.

---

## 📁 Project Structure

```
snake-game/
└── snake_game.html   # The entire game in a single file
```

---

## 🕹️ How to Play

1. Clone or download the repository
2. Open `snake_game.html` in your browser
3. Use the **arrow keys** to control the snake
4. Eat the red food to grow and earn points
5. Avoid hitting the walls or yourself
6. When the game ends, press **any key** to restart

---

## ⌨️ Controls

| Key                           | Action     |
| ----------------------------- | ---------- |
| `↑` Arrow Up                  | Move up    |
| `↓` Arrow Down                | Move down  |
| `←` Arrow Left                | Move left  |
| `→` Arrow Right               | Move right |
| Any key (on Game Over screen) | Restart    |

---

## ✨ Features

- Single-file game — zero setup required
- Smooth directional input with no accidental reverse
- On-canvas Game Over screen (no intrusive pop-ups)
- Live score tracker
- Restart without refreshing the page

---

## 🛠️ Built With

- HTML5 Canvas
- JavaScript
- CSS3

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/professional12022007/snake-game.git

# Navigate into the folder
cd snake-game

# Open the game
open snake_game.html
```

Or simply double-click `snake_game.html` — no server needed.
